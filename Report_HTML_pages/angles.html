Code
bool angle(Point p1, Point p2, Point p3)
{
float v1x = p1.x-p2.x;
float v1y = p1.y-p2.y;
float v2x = p3.x-p2.x;
float v2y = p3.y-p2.y;
float cross = v1x * v2y - v1y * v2x;
if (cross>=0)return true;
return false; }

Explanation

This code is used to calculate the angle formed by three points. It takes three points as arguments: p1, p2, and p3, and returns a boolean value. It does this by first calculating the vectors from p1 to p2 and from p3 to p2. It then takes the cross product of these two vectors and checks if it is greater than or equal to 0. If it is, it returns true, indicating that the angle formed is obtuse or right. If the cross product is not greater than or equal to 0, it returns false, indicating that the angle formed is acute.
