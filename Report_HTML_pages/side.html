Side

Code
int side(const Point &p1, const Point &p2, const Point &p3)
{
float val = (p2.y - p1.y) * (p3.x - p2.x) - (p2.x - p1.x) * (p3.y - p2.y);
if (val >= 0)
return 0;
else
return -1;
}

Explanation

This code is used to determine the orientation of three points, p1, p2, and p3. The function returns 0 if the points are in a clockwise orientation and -1 if they are in a counter-clockwise orientation. It does this by calculating the cross product of the two vectors that span the points.
